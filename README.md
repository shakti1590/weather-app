#### Problem statement:
 - Create a Weather app using React.js that allows users to view temperature.

#### Technology Used:
 - React Js
 - Java Script
 - OpenWeather Api



 ## Install

To install all the dependencies of the project, run the following command:

    git clone https://github.com/shakti1590/weather-app.git
    npm install
    npm start
